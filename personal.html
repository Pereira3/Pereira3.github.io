Unknown

Unknown is a game created as a final graduation project.
The idea for Unknown was to create a simple, easy-to-understand game that would allow anyone of any age to play. It is a platformer game, which consists of passing several levels with the aim of discovering the character's story.

The game build was published on Itch.io: Unknown

Were used Unity as a game engine (C# as a technology), VSCode as IDE and Itch.io and Unity Asset Store as asset source.
